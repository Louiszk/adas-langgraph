Based on a question provided under the state key 'question', the system should gather information online.
The final answer should be saved under the state key 'answer', and the relevant webpages must be saved under the state key 'sources' as a list of tuples `(title, link)`.
The system should break the query into its relevant parts, search those parts, and scrape linked pages when helpful.
It should always answer truthfully, including cases where it cannot find sources or relevant information.
The system must be robust and remain generalizable to answer any specified question.
